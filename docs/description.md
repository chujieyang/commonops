### 项目介绍

该项目的目标是想做一个一站式的运维管理平台，将企业私有服务器资源与公有云资源进行集中管理后，通过流程的制定将日常繁复的运维工作
进行自动化，涵盖日常运维的大部分操作；同时简化运维工作的操作流程和步骤。

目前项目采用的技术体系为: React(前端) + Golang(后端), 使用前后端分离的方式进行开发。
