# CommonOps <small>1.0.0</small>

> 集中化的运维管理平台

- 企业公有云服务器资源管理以及通过 Web SSH 进行维护（目前仅支持阿里云）
- 企业私有服务器资源的管理以及通过 Web SSH 进行维护
- Kubernetes 集群的资源管理、日志查看和远程命令执行
- 日常工单协作功能
- 使用DMS对用户进行权限设置后进行Mysql、PolarDb、SqlServer 的增删改查操作
- RBAC 用户功能权限设置
